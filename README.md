# Cloud Disk `云盘`

@(T Cloud)[时间:|2018/04/30|版本:|v1.0|撰写人:|July Time]

**T Cloud**个人文件云存储，文档同步云端，实时管理，方便快捷。
- **上传文件** ：支持图片，文档，音乐，视频等等；
- **文件查看** ：通过虚拟的文件夹，分类查看，可以进行在线管理，整理文档。
- **标签** ： 文件添加标签，收藏到同一个标签下，方便快捷，收藏管理。
